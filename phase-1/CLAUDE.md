# Instructions for Claude Code

This document provides instructions for using Claude Code to develop this application.

## Development Approach

*   Follow the Spec-Driven Development approach with Spec-Kit Plus.
*   Use Claude Code to generate code based on the specifications and plans.
*   Write specifications and plans in markdown files.
*   Break down the plan into specific tasks.
*   Implement the tasks accordingly.

## File Structure

*   `Constitution.md`: Project constitution.
*   `specify.md`: Project specifications.
*   `plan.md`: Project plan.
*   `tasks.md`: List of tasks.
*   `implement.md`: Implementation details.
*   `src/main.py`: Main application file.
*   `README.md`: Instructions on how to install and use the application.
*   `CLAUDE.md`: Instructions for Claude Code.

## Tasks

1.  Implement the data model (Task class).
2.  Implement the CLI commands.
3.  Implement the business logic.
4.  Write unit tests.
5.  Write documentation.
